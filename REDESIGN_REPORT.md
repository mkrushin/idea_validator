## 🎯 Дизайн-решения

### 1. Сигнатурный элемент: Numbered Sections
```html
<div class="section" data-number="01">
    <h3>Market Size & Trends</h3>
</div>
```

**Почему это работает:**
- Помогает пользователю ориентироваться в 8 разделах
- Создаёт ощущение структурированного "путешествия" через анализ
- Визуально отличает каждый раздел
- Работает как якорь при навигации

### 2. Палета: Emerald + Navy + Amber
**Психология цветов:**
- **Navy (#0F172E):** Доверие, профессионализм, серьёзность
- **Emerald (#00D084):** Валидация, успех, энергия (центральная идея)
- **Amber (#FF9F43):** Внимание, риски (вторичный акцент)

**Почему не другие варианты:**
- Не использованы в 90% других стартап-приложений → оригинально
- Зелёный означает "valid" в UI patterns → правильная ассоциация
- На тёмном фоне зелёный выглядит ярче и дороже

### 3. Темный режим как основной
**Выбор:**
- Дизайн оптимизирован для темного (не светлого как опция)
- Браузер может переопределить через `prefers-color-scheme`

**Почему это смелое решение:**
- Большинство приложений делают светлое основным
- Для tech-аудитории (предпринимательство) темный — стандарт
- Снижает нагрузку на глаза
- Выглядит более premium

### 4. Типография: Inter + Fira Code
**Display:** Inter Bold 700-800 (современный, уверенный)  
**Body:** Inter Regular 400 (чистота, читаемость)  
**Utility:** Fira Code 400 (напоминает код, техничность)

**Обоснование:**
- Inter используется в Apple, Figma, Stripe → ассоциируется с quality
- Fira Code для цифр/метаданных → техничный вид
- Вместе передают "tech startup" идентичность

## ✨ Микровзаимодействия

### Buttons
```css
transition: all 0.3s ease;

&:hover {
    background: #00C875;
    transform: translateY(-2px);
    box-shadow: 0 12px 30px rgba(0, 208, 132, 0.25);
}
```

### Cards
```css
border: 1px solid var(--border);

&:hover {
    border-color: var(--bright-emerald);
    box-shadow: 0 20px 60px rgba(0, 208, 132, 0.1);
}
```

### Stars (Rating)
```css
transition: all 0.2s ease;

&:hover {
    color: var(--warm-amber);
    filter: opacity(1);
    transform: scale(1.1);
    text-shadow: 0 0 20px rgba(255, 159, 67, 0.5);
}
```

**Общий принцип:** Гладкие 0.3s переходы, поднятие/масштабирование при hover, тени для глубины.
